# dia2dic

디아블로2 관련 자료를 모으고 검색하는 React 기반 웹페이지입니다.

## 기술 스택

- React
- TypeScript
- Vite
- React Router
- lucide-react

## 실행

```bash
npm install
npm run dev
```

## 스크립트

- `npm run dev`: 개발 서버 실행
- `npm run build`: 타입 체크 후 프로덕션 빌드
- `npm run lint`: ESLint 검사
- `npm run preview`: 빌드 결과 미리보기

## 초기 구조

- `/`: 검색 중심 홈
- `/items`: 아이템 자료
- `/runewords`: 룬워드 자료
- `/skills`: 스킬 자료
- `/areas`: 지역 자료
- `/systems`: 시스템 자료
- `/guides`: 가이드
